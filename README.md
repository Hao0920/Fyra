# Fyra

Fyra 是一个面向 iPad 风格仪表盘的记账前端原型，参考了 Moze iPad 设计。当前实现基于 React + Vite + Tailwind，目标是先完成前端界面，再逐步加入数据同步和账本功能。

## 运行

1. 安装依赖
   ```bash
   pnpm install
   ```
2. 本地开发
   ```bash
   pnpm dev
   ```

## 目录结构

- `src/` - 应用源码
- `src/components/` - UI 组件
- `src/data/` - 模拟数据
- `src/types.ts` - 类型定义
